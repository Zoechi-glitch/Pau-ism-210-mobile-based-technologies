# Eboka Mirabel first launched backend API

This is a backend API for handling login and form submissions.
 Built with Node.js, Express, and MongoDB then hosted with Render

 # Installation
 1. copy repo link

 2. Install dependencies 
 npm install

 3. Start the server 
 npm start

 # usage
 - Post / api/submit-form/login
 - Send JSON data in request body
